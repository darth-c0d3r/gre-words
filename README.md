## How to Run?
Just run ```python3 practice.py```. It will read the words from words.txt and ask you sequentially. It has various options like shuffle and reverse. Once you are gone through all the words or discontinue practice, it will repeat all the words that you got incorrect in the practice. This cycle keeps running iteratively.
